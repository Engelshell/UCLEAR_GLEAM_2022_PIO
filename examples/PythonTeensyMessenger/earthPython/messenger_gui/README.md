
Requires:
* Python 3
* USB to UART
* Configure Xbee in 802.15.4 transparency mode

dependencies:
```
pip install pyside6
pip install cobs
pip install msgpack
```


run
```
python example_gui_messenger.py
```


mainwindow.ui made using Qt Designer (Not design studio). The .ui file outputs the example_gui_messenger_ui.py.
